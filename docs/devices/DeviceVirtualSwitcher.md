# Virtual Switcher

This is an experimental device to allow sending the content of one of the input channels to the output channel.

## Device configuration settings

  - **DeviceType**: (`Type`) = `(`VirtualSwitcher`)` (Required)
  - **DeviceAcquisitionRate**: (`AcquisitionRate`) (Optional, default: 50) 
  - **LocalTimeOffsetSec**: (Optional, default: 0)
